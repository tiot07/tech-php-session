# ①課題番号-プロダクト名

食事管理アプリ

## ②課題内容（どんな作品か）
- 現在地から近い場所にある食べ物の一覧を表示して、記録できる。
- ログイン機能、ログアウト機能、ユーザ新規作成機能を取り入れました。
- 特に管理ユーザのみの機能はないですが、ログインした後の画面でどちらのユーザかを表示するようにしました。
